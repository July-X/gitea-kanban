P0-2 · 列 = label UI 显式标注

拍板 2026-06-16：column.labels[] 数据已在 schema / store，但 UI 现状是"label 列表 join 成一行 + 统一灰色 icon"——user 看见不到"列绑了哪些 gitea label、为什么这些 issue 进这列"。

这次把"列 = label"这件事从隐式变显式：列头 chip 化用 gitea label 真实颜色、列没绑 label 时显式红边警告、列菜单批量建列 + 自动绑剩余 label 入口。

场景 3：列没绑 label → 红边警告

列绑的 label 集合为空（user 手动建了一列 + 没绑任何 label），或者绑的 label 都被删了——这种情况 user 不知道为什么"issue 进不来"。红边虚线 + banner 明确告知 + 一键跳到列设置。
