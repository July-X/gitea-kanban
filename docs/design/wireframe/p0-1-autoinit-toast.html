! 列和 label 是怎么关联的？ ×
列匹配 gitea 仓库里的 label（不是列自己存卡片）。 改列绑的 label → 这列里的 issue 跟着变。

设计要点： ① 不弹 autoInit toast（避免"系统啥都没干却弹个窗"）
② EmptyState 文案要 教：列和 label 的关系（v1 现状文案没写，user 看了还是不懂）
③ "+ 新增列" 按钮走列创建向导（不是裸弹个模态），向导里有"建列 + 绑 label"二合一
④ 黄色 toast 是 概念解释——"列匹配 gitea label"这件事 user 第一次用根本不知道
