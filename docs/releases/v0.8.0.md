## 零术语合规

| 原词 | UI 文案 |
|---|---|
| update / check for updates | 更新 / 检查更新 |
| download | 下载 |
| install / restart | 安装 / 重启以安装 |
| release notes | 更新日志 |
| manifest / signature / canary | 不出现（内部概念） |
| macOS unsigned manual path | 「前往下载页」（macOS 未签名 build 显示） |

## 红线维持

- ❌ 不引入 `electron-updater` 等 Electron 生态工具
- ❌ 不引入 `minisign` 等新三方依赖（ed25519 走 Go 标准库）
- ❌ 不支持 Linux 平台（v0.8.0 仅 stable + Windows/macOS 双平台）
- ❌ 不支持 canary 多通道
- ❌ 不做 NSIS / Inno Setup 等第三方安装包
- ❌ 不在前端 `v-html` 渲染 release notes（必须走 markdown-it + dompurify，SettingsView 实现时）
- ❌ 不阻塞 UI 启动（checkUpdatesAtStartup 走 goroutine + 静默降级）

## 不做清单（v0.8.0 范围外）

- Windows Authenticode 签名
- macOS Notarization 证书流程（applyMacOS 仍走 OpenDownloadPage）
- canary 多通道分发
- 差量级热更新（block-level patch）
- R2 / 自建 CDN mirror（GitHub release 作唯一源）
- security_review 4 个 medium 修复（GitHub/Gitea adapter 错误 body 直传 + JSON decode 无 size limit；属 v0.7.x 既有 bug，留待后续专门 commit）
