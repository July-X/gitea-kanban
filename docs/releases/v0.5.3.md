# v0.5.3 Release Notes

> **发布日期**：2026-07-07
> **基于**：v0.5.0 + v0.5.1-v0.5.3 累积修复 + 开发中未发版改动（搜索/主题/清理）
> **相关 ADR**：[ADR-0008](./docs/adr/0008-pr-comment-v05-enhancement.md)

## 1. 版本定位

v0.5.3 是 v0.5.0 发版后的累积优化版，主要聚焦在：
- **Git Graph 视觉与交互打磨**（亮点：提交搜索、亮色主题、滚动加载行为）
- **App 性能与稳定性**（StatusBar 按钮防抖、雪屏修复、路由守卫优化）
- **代码健康**（dead code 清理 -803 行、错误码瘦身、platform 接口收敛）

> ⚠️ 以下 v0.6.x 描述的改动仍在 main 分支开发中，**未包含在 v0.5.3 发版**。

## 2. 发版范围

### 2.1 Git Graph 提交搜索

| 项 | 说明 |
|---|---|
| 新组件 | `GitGraphFindWidget.vue` — 搜索框 UI，含正则/大小写、上下导航、匹配计数、Escape 关闭 |
| 新接口 | `SearchState` interface + `search-change` emit，向父组件暴露匹配集 + 正则 pattern |
| 后端 | fix `watch` 去掉 `deep:true`（避免 graphDto 内部变化触发重复遍历） |
| 前端状态 | `searchState` ref + `onSearchChange` handler |
| 视觉双层反馈 | `commit-row--search-match`（匹配但非当前选中 → 渐变背景）+ `commit-subject__find-match` span（黄色文字高亮 + 细边框）+ 行渐变 + outline |
| 对齐 | vscode-git-graph（2026-07-07 原版行为） |

### 2.2 亮色主题最终方案

| 项 | 旧值 | 新值 |
|---|---|---|
| 背景 `--color-canvas` | `#E8F1F5`（冷蓝灰） | `#FFFFFF`（纯白） |
| 浮动层 `--color-elevated` | `#FFFFFF` | `#F8FAFC`（极浅灰白） |
| 主色 `--color-primary` | `#466B16` (暗) / `#0F766E` (亮) | `#466B16`（**亮暗统一 Gitea 绿**） |
| 文字 `--color-text` | `#000000` | `#0F172A`（slate-900） |
| 次文字 `--color-text-muted` | `#475461` | `#475569`（slate-600） |
| 分隔线 `--color-divider` | `rgba(15,26,36,0.1)` | `rgba(15,26,36,0.08)` |

**设计思路**：亮色主题不再强调"冷蓝底"，改为 Linear/Notion 风格 — 纯色画布 + 极浅灰白浮动层 + 主色 Gitea 绿统一锚定。WCAG AA 对比度全程 ≥ 4.5:1。

### 2.3 滚动加载行为修正

| 问题 | 旧行为 | 新行为 |
|---|---|---|
| 加载后位置跳动 | `scrollTop = savedTop + addedHeight`（跳到新数据末尾） | 对齐 vscode-git-graph `loadMoreCommits` — 不主动调 scrollTop，用户继续从原有视线位置往下翻 |

### 2.4 App 性能与开发体验

| 项 | 说明 |
|---|---|
| 路由守卫 | 只读同步 `auth.isConnected`，禁止 `await auth.refreshStatus()` |
| Suspense 包裹 | `<router-view>` 包 `<Suspense>` + 骨架屏 fallback，替代懒加载期白屏 |
| 防抖 | StatusBar 刷新按钮 800ms 防抖，fire-and-forget（`void doRefresh()`） |
| console 拦截 | 轻量 `fastSerialize`，不 JSON.stringify 大对象，防循环引用冻屏 |
| Wails HMR | `vite.config.ts` 加 `server.allowedHosts` + `hmr.host`，修复 `wails dev` WebSocket 连接失败 |
| ponytail-audit 清理 | -803 行死代码（`WorkspaceMigrateDialog.vue` 整文件、workspace migration IPC 助手、Go 后端 dead export 等） |
| 平台错误码 | 删除 4 个 unreachable error codes（theme_not_found / invalid_theme / database_unavailable / database_write_failed） |
