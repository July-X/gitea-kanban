> **TL;DR**：user 反馈 3 个 PR 对话区 UI 调整 —— 移除 `pr-detail__conv-header` 标题 div / timeline icon 放大 / 文字升一档。v0.7.10 全部改完。

## 范围

### 前端（1 个 commit）

- **`feat(v0.7.10): PR 对话区 UI 微调`** —— frontend/src/views/MergesView.vue
  3 类改动（移除对话标题 div + timeline dot/icon 放大 + 文字升一档）

## 改动详情

### 1. 移除 `pr-detail__conv-header` 整块

整 div 删除（含 MessageSquare icon + "对话" 文字 + 计数 badge + 刷新按钮）：

```vue
<!-- 删除前 -->
<div class="pr-detail__conv-header">
  <div class="pr-detail__conv-header-left">
    <MessageSquare :size="14" :stroke-width="2" aria-hidden="true" />
    <span>对话</span>
    <span v-if="getTimelinePanel().items.length > 0" class="pr-detail__conv-count">
      {{ getTimelinePanel().items.length }}
    </span>
  </div>
  <button
    type="button"
    class="btn-ghost-sm"
    :disabled="getTimelinePanel().loading"
    title="刷新对话"
    @click="fetchComments(selectedPR)"
  >
    <RefreshCw :size="12" :stroke-width="2" aria-hidden="true" />
    <span>刷新</span>
  </button>
</div>
```

删除原因：user 反馈"对话标题这个 div 移除，不需要展示出来" —— 对话 tab 已经显示在 PR header 工具栏（"对话 13 / 代码提交 1 / 文件变动 1"），再重复一次"对话"标题是冗余。

CSS 同步删 `.pr-detail__conv-header / .pr-detail__conv-header-left / .pr-detail__conv-count`（避免样式死代码）。

刷新按钮如果后续需要可以走 tab 切换 / PR header 工具栏 / 快捷键，不影响 timeline 渲染。

### 2. Timeline icon 放大

`.pr-detail__timeline-dot` 从 22px 改 26px（4px 增量，"大一点点"），内部 icon size 13 → 15 同步放大（review state icon + system event icon 2 处）保持视觉比例。

| 元素 | 改前 | 改后 |
|---|---|---|
| `.pr-detail__timeline-dot` width/height | 22px | 26px |
| review state icon (CheckCircle2/XCircle/MessageCircle) size | 13 | 15 |
| system event icon (GitMerge/Tag/UserPlus/...) size | 13 | 15 |

### 3. Timeline 文字字号升一档

```css
/* 改前 */
.pr-detail__event-line     { /* 无显式 font-size，继承默认 13px */ }
.pr-detail__event-prep     { font-size: var(--font-sm);  /* 13px */ }
.pr-detail__event-time     { font-size: var(--font-xs);  /* 11px */ }
.pr-detail__event-inline   { font-size: var(--font-sm);  /* 13px */ }

/* 改后 */
.pr-detail__event-line     { font-size: var(--font-body); /* 14px +1px */ }
.pr-detail__event-prep     { font-size: var(--font-body); /* 14px +1px */ }
.pr-detail__event-time     { font-size: 12px;             /* 12px +1px（inline 不引 --font-mono） */ }
.pr-detail__event-inline   { font-size: var(--font-body); /* 14px +1px */ }
```

不引 `--font-mono`（12px 等宽字体）：event-time 显示"X 天前"中文 + 数字，用 mono 字体会怪，用 inline 12px 更合适。

## 影响

- 对话区更干净（无重复标题）
- timeline icon 视觉重量提升（跟 Gitea web 28px 接近，差 2px）
- timeline 文字更易读（13px → 14px，跟 PR header / description 区域字号对齐）
