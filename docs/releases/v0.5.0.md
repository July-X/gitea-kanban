# v0.5.0 Release Notes

> **发布日期**：2026-07-04（planned）
> **Milestone**：M4 — PR 评论模块对齐 Gitea/GitHub
> **相关 ADR**：[ADR-0008](./docs/adr/0008-pr-comment-v05-enhancement.md)

---

## 1. 本次发布范围

v0.5.0 把 gitea-kanban 的 PR 评论模块从「列评论+发评论」升级为对齐 Gitea 1.21+ 与 GitHub REST v3 的完整评论能力，覆盖 4 大能力域、6 个新平台接口、4 个前端新组件。

---

## 2. 后端变更（Go）

### 2.1 PlatformAdapter interface 扩展

| 新增方法 | 签名 | 说明 |
|---|---|---|
| `ListPullReviewComments` | `(ctx, host, user, token, owner, repo, index) → ([]PullReviewCommentDto, error)` | 按 PR 拉行内评审评论 |
| `CreatePullReviewComment` | `(ctx, host, user, token, owner, repo, index, body, path, line) → (*PullReviewCommentDto, error)` | 发行内评审评论 |
| `ListPullFiles` | `(ctx, host, user, token, owner, repo, index) → ([]PullFileDTO, error)` | PR 变更文件列表 |
| `GetPullFileDiff` | `(ctx, host, user, token, owner, repo, index, filePath) → (*PullFileDiffDTO, error)` | 单文件 unified diff |

### 2.2 Gitea / GitHub 双平台实现

| 能力 | Gita 实现 | GitHub 实现 | 兼容性 |
|---|---|---|---|
| ListPullReviewComments | GET /pulls/{index}/comments | GET /pulls/{number}/comments | 全版本 |
| CreatePullReviewComment | POST + `{body, path, new_position}` | POST + `{body, path, line}` | 全版本 |
| ListPullFiles | GET /pulls/{index}/files | GET /pulls/{number}/files | 低版本返 `ErrNotSupported` |
| GetPullFileDiff | GET /pulls/{index}.diff 前按文件拆分 | GET /pulls/{number}` + Accept: diff | 低版本返 `ErrNotSupported` |

### 2.3 App bindings（Wails）

| 新方法 | 入参 | 出参 |
|---|---|---|
| `App.ListPullReviewComments` | `{projectId, index}` | `PullReviewCommentDto[]` |
| `App.CreatePullReviewComment` | `{projectId, index, body, path, line}` | `PullReviewCommentDto` |
| `App.ListPullFiles` | `{projectId, index}` | `PullFileDTO[]` |
| `App.GetPullFileDiff` | `{projectId, index, filePath}` | `PullFileDiffDTO` |

### 2.4 DTO 扩展

| 新 DTO | Go 字段 | 前端 TS | 用途 |
|---|---|---|---|
| `PullReviewCommentDto` | `id, body, author, path, line, createdAt, updatedAt` | 同名 | 行内评论 |
| `PullFileDTO` | `filename, status, additions, deletions, changes, patch, previousFilename` | `PullFileDto` (camelCase) | 文件变更 |
| `PullFileDiffDTO` | `filename, rawDiff, hunks[]` | `PullFileDiffDto` | 单文件 diff |
| `PullDiffHunk` | `oldStart, oldLines, newStart, newLines, header, lines` | `PullDiffHunkDto` | Hunk 块 |

### 2.5 Go 单测

- `app/platform/gitea/adapter_test.go` — 添加 httptest mock（新增 4 个测试）
- `app/platform/github/adapter_test.go` — 同上（新增 4 个测试）
- `app/platform/adapter_test.go` — 接口兼容性编译验证

## 3. 前端变更（Vue 3 + TypeScript）

### 3.1 DTO + IPC

| 新文件 | 内容 |
|---|---|
| `frontend/src/types/dto.ts` +3 interface | `PullFileDto`, `PullFileDiffDto`, `PullDiffHunkDto` |
| `frontend/src/lib/ipc-client.ts` +4 方法 | `pullsReviewCommentsList`, `pullsReviewCommentCreate`, `pullsFilesList`, `pullsFileDiffGet` |

### 3.2 Store 扩展

`frontend/src/stores/pull.ts` 重构：

- 新增 state：`reviewCommentsByPR`, `filesByPR`, `fileDiffByPath`
- 新增 action：`loadReviewComments`, `loadFiles`, `loadFileDiff`, `submitReview`, `fetchReviews`
- 新增 getter：`timelineItems`（按 issue 评论 + review 事件时间升序合并）
- 新增面板：评论面板 `commentPanels`、评审面板 `reviewPanels`

### 3.3 组件新增

| 新组件 | 行数 | 用途 |
|---|---|---|
| `PullFileComments.vue` | 383 | 按文件分组，折叠展开，行内评论 |
| `ReviewEventCard`（内联 MergesView） | ~60 | 对话流中的系统消息卡片 |

### 3.4 MergesView.vue 改造

- PR 详情顶部新增三 Tab 开关：**概览 / 文件评论 / 对话**
- 概览 Tab：原 meta + 评审区（维持 v1.x 信息架构）
- 文件评论 Tab：渲染 `<PullFileComments>` 组件
- 对话 Tab：按 `timelineItems` 渲染混合时间线

### 3.5 样式新增

| 类名 | 作用 |
|---|---|
| `.merge-item__detail-tabs` | Tab 行容器 |
| `.merge-item__detail-tab--active` | 当前 Tab 高亮 |
| `.merge-item__comment--review-event` | 评审卡片虚线弱化 |
| `.merge-item__comment-avatar--approved/changes_requested/commented` | 状态 icon |

## 4. 已知限制

| 限制项 | 影响 |
|---|---|
| GitHub / Gitea 低版本不支持 pulls/files | 前端隐藏文件评论 Tab |
| Review 事件卡片暂无撤销按钮 | 后续迭代 |
| 文件 diff 不渲染行内 Suggested Change | P2 功能 |
| 单 PR 评论 + reactions + reviews 一次性并发拉取 | PR < 50 条时无性能问题 |

## 6. 迁移指南

本次改动**无数据迁移**。前端升级后建议：

```bash
# 后端
go mod download
go build -o /dev/null .

# 前端
cd frontend
pnpm install
pnpm build

# 三端构建
wails build
```
