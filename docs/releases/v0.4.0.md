## 三大主功能（用户拍板）

### 1. Git Graph 加载更多 UI 收敛

GitHub 仓库 UI 顶部「加载更多」按钮 + 滚动监听彻底删除，与 Gitea UI 对齐（含后端 `App.DepenRepo` binding、`app/git/deepen.go`、`frontend/src/lib/ipc-client.ts` 的 `deepenRepo` 全栈清理）。Graph DTO `truncated` 字段保留兼容，前端不会再触发手动加载更多。

关键 commit：

- `85e63a8 feat(gitgraph): 彻底删除「加载更多」按钮 + 滚动监听 + 后端 DeepenRepo 链路`

### 2. StatusBar 顺序调整

仓库 dropdown 与 api URL 位置对调，最终顺序 `chip → api URL → 仓库 dropdown → 刷新仓库 → 主题`。

关键 commit：

- `dc004ad feat(statusbar): 调整仓库 dropdown 与 api URL 位置对调`
- `460f46c fix(statusbar): 真正把 api URL <span> 上移到仓库 dropdown 之前`

### 3. Git 二进制内嵌（macOS + Windows）

默认应用内嵌 `git 2.55.0` 二进制（`go:embed` 到 Go binary，`${dataDir}/tools/git/git-2.55.0-${GOOS}-${GOARCH}[.exe]` 释放），Linux 平台不嵌入（系统 PATH 足够）。新增 `app/gitbinary` 包统一 Runner 抽象：

```
RunGit / RunGitWithEnv / ResolveGitBinaryPath / TestGitBinary / Init
```

所有 11 处 `exec.Command("git", ...)` 生产调用点改走 Runner。**用户可在 SettingsView「Git 二进制」卡片自定义路径**（macOS / Windows / Linux 都支持，平台特定文件选择对话框 + `git --version` 验证 + macOS Gatekeeper `xattr -d com.apple.quarantine` 自动剥离）。LocalState 新增 `prefs["app.gitBinaryPath"]` 字段，进程内 `gitbinary.SetUserOverride` 立即生效无需重启。

关键 commit：

- `18beec3 feat(gitbinary): 内嵌 git 2.55.0 二进制 + 用户可配置路径`
- `271fc3b feat(settings): 设置页新增「Git 二进制」卡片 + settings store 加 gitBinaryPath 字段`
- `e7a5344 test(gitbinary): 修 file name 隐式 GOOS constraint + 补 macOS quarantine 测试`
- `91b17bd fix(gitbinary): 内嵌 binary 文件名不能以 git- 开头 + PATH git 优先级提到 embedded 之前`
- `97aa7f9 feat(settings-gitbinary): v0.5 2-button 模式选择 UI（user-mid-turn steer）`
- `126392a feat(settings-gitbinary): v0.5-mid2 砍路径/高级/已释放+改「系统装的Git」为弹窗选 file`
- `58c2573 feat(gitbinary): 「使用系统装的 Git」弹窗初始目录偏好系统 PATH git 所在目录`
- `ec64848 feat(gitbinary): 弹窗初始目录解析 symlink 到 git 实体 bin`
- `cec21dc fix(gitbinary): macOS 文件选择器 filters 让 Unix executable 也能选`

## v0.4.1 字体热修复（分支中段先发）

> 注：`v0.4.1` 是分支中段针对字体渲染问题紧急放出的 patch release（annotated tag @ `60755d4`）。tag 顺序在 v0.4.0 之前打，但 v0.4.0 release 自然包含 v0.4.1 的全部变更。

修复点：

- **macOS 2K 屏文字发虚**：移除 `-webkit-font-smoothing: antialiased` / `-moz-osx-font-smoothing`，恢复系统默认子像素抗锯齿
- **等宽字体内嵌**：JetBrains Mono + MesloLGS NF，防止用户系统缺字体
- **默认字号上调**：14px→15px / 13px→14px / 12px→13px
- **Git Graph 字体对齐 vscode-git-graph 官方**：commit row/detail 13px，ref badge 12px
- **暗色主题文字提亮**：`--color-text` 改为纯白 `#ffffff`，`--color-text-muted` `#b4c6d4`
- **等宽字体栈更新**：JetBrains Mono → Consolas → MesloLGS NF，系统字体移至末尾兜底

关键 commit：`4688f29` / `8435ab0` / `4373999` / `7e0d348` / `120b170` / `a95de6d` / `c6467ad`
